Acordeón

Un componente de UI es una pieza de interfaz de usuario autocontenida y reutilizable. Puede ser un botón, un campo de texto o un componente más complejo como un selector de fechas. El objetivo es construir interfaces complejas a partir de bloques de construcción simples.
Técnicas como variables CSS, Flexbox, Grid y animaciones permiten crear sistemas de diseño robustos, mantenibles y escalables. Fomentan la reutilización, la consistencia visual y hacen que el desarrollo sea más rápido y eficiente.
Sí, este componente de acordeón está diseñado teniendo en cuenta la accesibilidad. Utiliza el atributo `aria-expanded` para comunicar el estado de cada sección a los lectores de pantalla y es completamente navegable por teclado.
